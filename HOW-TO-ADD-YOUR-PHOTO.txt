üì∏ RECOMMENDED PHOTO SPECIFICATIONS
--------------------------------------------------------------------------------

IDEAL DIMENSIONS:
- Minimum: 800px x 800px
- Recommended: 1200px x 1200px
- Maximum: 2000px x 2000px
- Aspect Ratio: 1:1 (Square)

FILE FORMAT:
- JPG/JPEG (best for photos)
- PNG (if you need transparency)
- WebP (best performance)

FILE SIZE:
- Keep under 500KB for optimal performance
- Use image compression tools like TinyPNG or Squoosh

PHOTO STYLE:
- Professional headshot or portrait
- Good lighting
- Clean background (solid color or subtle gradient)
- Face clearly visible
- Centered composition

üé® STYLING OPTIONS
--------------------------------------------------------------------------------

The photo container already has:
‚úì Rounded corners (rounded-3xl)
‚úì Gradient background
‚úì Border
‚úì Shadow effects
‚úì Blur animations around edges

OBJECT-FIT OPTIONS (if photo doesn't look right):
- object-cover: Fills the container, crops if needed (DEFAULT - RECOMMENDED)
- object-contain: Shows entire photo, may have empty space
- object-fill: Stretches to fill (can distort)

OBJECT-POSITION OPTIONS (to adjust photo positioning):
- object-center: Centers the photo (DEFAULT)
- object-top: Focuses on top of photo (good for headshots)
- object-bottom: Focuses on bottom
- object-left: Focuses on left side
- object-right: Focuses on right side

Example with custom positioning:
   className="object-cover object-top"


üîß TROUBLESHOOTING
--------------------------------------------------------------------------------

PHOTO NOT SHOWING?
- Check file path is correct (case-sensitive!)
- Make sure photo is in /public folder
- Verify file extension matches (.jpg vs .jpeg vs .png)
- Clear browser cache (Ctrl + Shift + R)
- Check browser console for errors (F12)

PHOTO LOOKS STRETCHED OR CROPPED WEIRDLY?
- Try object-contain instead of object-cover
- Adjust object-position (see styling options above)
- Ensure your photo is square (1:1 aspect ratio)

PHOTO LOADS SLOWLY?
- Compress your image (use TinyPNG or Squoosh)
- Convert to WebP format
- Ensure file size is under 500KB
